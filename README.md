# Data Analysis of Indian Banks (Jan–Aug 2025)

## Overview
This Excel project analyzes Indian banks and banking infrastructure using RBI monthly data from **January 2025 to August 2025**. It provides interactive dashboards to highlight banking performance, digital usage trends, and infrastructure insights.

## Data Source
Reserve Bank of India (RBI) — Monthly banking statistics (Jan–Aug 2025)  
https://www.rbi.org.in/Scripts/ATMView.aspx?atmid=170  

## Key Insights / Dashboards
- KPI Dashboard: transaction value, penetration rate, cash dependency, avg. transaction value  
- Credit vs Debit transaction analysis  
- Top 10 Banks ranking dashboards  
- Infrastructure dashboard and MoM trends  

## Tools Used
- Excel Pivot Tables  
- Pivot Charts  
- Slicers  
- Power Query  
- Conditional Formatting  

## Usage
1. Open the Excel file  
2. Go to **Data → Refresh All** to update dashboards  
3. Use slicers to explore insights
